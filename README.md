# Nect Cloud Projekt - Modul 346


# Kurzbeschreibung
In diesem README wird beschrieben wie man das script erfolgreich ausführen kann und somit auch die Infrastruktur erfolgreich erstellt wird

# Installation Guide

## Schritt 1 
Zuerst Clonen wir das Git repository auf unseren PC
```
git clone https://github.com/deonramadani/M346-Projekt.git
```

## Schritt 2
Danach wechseln wir in den Projekt-Ordner
```
cd M346-Projekt
```

## Schrit 3
Damit wir die Datei ausführen können müssen wir die berechtigung setzen
```
chmod +x deploy.sh
```

## Schritt 4
Wenn die berechtigung gesetzt wurde kann das Scipt ausgeführt werden
```
./deploy.sh
```

## Nach dem ausführen
Nach dem ausführen des deploy.sh sollte ca. 5 Minuten gewartet werden bis alle Installationen durchgeführt worden sind.
